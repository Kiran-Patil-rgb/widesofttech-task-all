  * string manipulation *
  defination:
 String manipulation in JavaScript refers to the various techniques and methods used to modify, extract, combine, or analyze textual data represented as strings. 
 Although strings in JavaScript are immutable
  (meaning their content cannot be directly changed after creation), manipulation methods return new strings with the desired modifications.

  * Common string manipulation techniques and methods include:
 1> Concatenation: Combining two or more strings using the + operator or the concat() method.
 ex,
 let greeting = "Hello";
    let name = "World";
    let message = greeting + " " + name + "!"; // "Hello World!"
    let anotherMessage = greeting.concat(" ", name, "!"); // "Hello World!"
  2> Accessing Characters: Retrieving individual characters within a string using bracket notation [] or the charAt() method.
  let text = "JavaScript";
    let firstChar = text[0]; // "J"
    let charAtIndex = text.charAt(4); // "S"

  3> Slicing and Substrings: Extracting portions of a string using slice(), substring(), or substr().
   ex,
   let sentence = "The quick brown fox";
    let part1 = sentence.slice(4, 9); // "quick"
    let part2 = sentence.substring(10, 15); // "brown"
    let part3 = sentence.substr(16, 3); // "fox"

    4>Case Conversion: Changing the case of characters using toUpperCase() and toLowerCase().
    ex,
     let mixedCase = "HeLlO WoRlD";
    let upper = mixedCase.toUpperCase(); // "HELLO WORLD"
    let lower = mixedCase.toLowerCase(); // "hello world"

    5>Searching and Replacing: Finding specific substrings and replacing them using indexOf(), includes(), startsWith(), endsWith(), replace(), and replaceAll().
    ex,
    let phrase = "Learn JavaScript";
    let found = phrase.includes("Java"); // true
    let newPhrase = phrase.replace("JavaScript", "Node.js"); // "Learn Node.js"

    6> Trimming Whitespace: Removing leading or trailing whitespace using trim(), trimStart(), and trimEnd().
    ex,
     let paddedString = "  Hello  ";
    let trimmed = paddedString.trim(); // "Hello"

    7> Template Literals: A modern way to create and manipulate strings, allowing for embedded expressions and multi-line strings using backticks `` ` ``.
      let item = "book";
    let price = 19.99;
    let details = `The ${item} costs $${price}.`; // "The book costs $19.99."
